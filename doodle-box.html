Doodle Box!

Doodle Box is a simple pixel art doodling tool.
To start doodling please select a Canvas Size.
You will be able to create a Palette of colors to doodle with on the left.
To use a color on your Palette simply click it then click the pixel you would like to color.
You can edit the colors in your palette by clicking the Edit Button.
To turn off the grid around each pixel click the Toggle Grid Tool.
To save the image you created to your computer click the Save Doodle Tool.
To clear the entire Canvas click the Clear Doodle Tool.
To erase a single pixel select the Eraser Tool and then click the pixel you wish to erase.
Thank you for using Doodle Box and enjoy!
